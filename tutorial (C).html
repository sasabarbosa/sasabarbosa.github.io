Passos para programar em C

Passo 01:

Baixe e instale um compilador.

Estude os comandos principais, como:

#include: Esse comando possibilita o carregamento das bibliotecas com todas as funções que o usuário vai precisar.

Main(): Esse comando é o que vai dar ínicio ao programa.

{}: Indica que tudo o que está dentro dele faz parte da função.

Printf(): Esse comando mostra o conteúdo presente dentro dele na tela do usuário, é necessário a utilização de aspas nesse comando.

Scanf(): Esse comando vai ler o conteúdo presente no printf.

;: Indica que é o fim da linha.

Dica final: Deixe comentários no código, assim qualquer um que ler o código de seu programa vai saber o que fazer.

Passo 02:

Utilize variáveis lógicas, nelas o usuário pode guardar dados.As variáveis são:

int (%d): Essa variável é utilizada para números inteiros.

float(%f): Essa variável é utilizada para números decimais.

char (%c): Essa variável é utilizada para representar caracteres.

Passo 03

Utilize variáveis condicionais (if e else).

O if e else são utilizados para comparar uma condição, se a condição for falsa irá realizar tal comando, mas se ela for verdadeira irá utilizar outro tipo de comando.O else só será executado se a condição if for falsa.

Para fazer essas comparações utilize operadores condicionais,como:

(>) maior que.

(<) menor que.

(>=) maior ou igual a.

(<=) menor ou igual a.

(==) igual a.

(!=) diferente.

Exemplo:

#include

main(){

float N1,N2,N3,N4,MD;

printf("Digite sua primeira nota:");

scanf("%f",&N1);

printf("Digite sua segunda nota:");

scanf("%f",&N2);

printf("Digite sua terceira nota:");

scanf("%f",&N3);

printf("Digite sua quarta nota:");

scanf("%f,&N4");

MD=(N1+N2+N3+N4)/4;

printf("Sua média é %f",MD)

}
